# Recommended-System
# 🎯 Рекомендательная система постов

**Задача:** Построить рекомендательную систему для персонализированной ленты постов  
**Метрика:** Hitrate@5  
**Требования:** <0.5 сек на запрос, <4 ГБ памяти  
---

## 📊 Описание задачи

**Проблема:** Показываем случайные посты → низкая релевантность  
**Решение:** Персонализированные рекомендации на основе:
- 📈 Активности пользователя
- 👤 Характеристик профиля
- 📄 Содержимого постов
- ⏱️ Временных факторов

---

## 🎯 Цель проекта

Разработать сервис рекомендаций, который для каждого пользователя в любой момент времени возвращает **5 наиболее релевантных постов** для показа в ленте.

---

## 📈 Метрика качества: Hitrate@5

$$
HR@5 = \frac{1}{n \cdot T} \sum_{t=1}^{T} \sum_{i=1}^{n} \min\left(1, \sum_{j=1}^{5} [a_j(x_i, t) = 1]\right)
$$

**Где:**
- $n$ — количество пользователей
- $T$ — количество периодов проверки
- $a_j(x_i, t)$ — j-я рекомендация i-му пользователю в момент времени t

**Проще говоря:** Доля пользователей, которым среди топ-5 рекомендаций попался хотя бы 1 релевантный пост.

###  2. Инженерия признаков
**Признаки пользователя:**
- Демография (возраст, пол, образование)
- Активность (количество лайков, просмотров)
- Предпочтения (любимые темы, сообщества)

**Признаки постов:**
- Текстовые (темы, категории, сложность)
- Метрики вовлеченности (лайки, просмотры)
- Временные (дата публикации, актуальность)

**Взаимодействия:**
- Матрица пользователь-пост
- Временные паттерны активности
